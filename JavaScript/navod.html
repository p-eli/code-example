Dokumentace - Tabulka s dynamickými sloupci a řádky

Jakub Pelikán - xpelik14


  • Použití skriptu:
    1. Vložení skriptu do hlavičky html souboru:
        <script src="dynamicTable.js"></script>
    2. Vložení volání funkce pro zadání ID tabulky na konec HTML dokumentu:
      • Manuální zadání ID tabulky pomocí funkce initDynamicTable:
          <script>initDynamicTable(id)</script>
      • Automatická inicializace, která zavolá funkci initDynamicTable pro všechny tabulky v dokumentu, které mají zadáno ID, pomocí funkce autoInitDynamicTables:
          <script>autoInitDynamicTables()</script>

  • Činnost skriptu:
    • Změna šířky sloupců a výšky řádků.
    • Změna velikosti celé tabulky, pomocí tažení za rohy tabulky.
    • Změna pořadí sloupců pomocí prohození libovolných dvou sloupců mezi sebou.
    • Změna pořadí sloupců pomocí přesunutí libovolného sloupce před/za jiný.
